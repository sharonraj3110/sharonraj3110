### 🧬 [Differential Gene Expression Analysis in Breast Cancer](./breast-cancer-rnaseq)
> RNA-seq · DESeq2 · R · ggplot2 · GEOquery

- Analyzed publicly available RNA-seq data (GSE183947) comparing breast cancer tumor vs. normal tissue
- Built a complete pipeline: data download → preprocessing → normalization → DESeq2 testing → visualization
- Identified 500+ significantly differentially expressed genes (padj < 0.05, |log2FC| > 1)
- Generated volcano plots, heatmaps, PCA plots, and pathway enrichment analyses
- Top findings: cell cycle progression (MKI67, TOP2A, CCNB1) and ECM remodelling pathways upregulated in tumor
